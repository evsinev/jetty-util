# Jetty utilities

Util classes for jetty


## How to add it into your app

### Maven


```xml
<repositories>
    <repository>
        <id>pne</id>
        <name>payneteasy repo</name>
        <url>https://maven.pne.io</url>
    </repository>
</repositories>
  
<dependency>
    <groupId>com.payneteasy</groupId>
    <artifactId>jetty-util</artifactId>
    <version>1.0-1</version>
</dependency>
```
